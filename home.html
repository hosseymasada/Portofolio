ABOUT NOKA 3

NoKa 3 is a 5v5 First Person Shooter game that has the motto "Competing With Yourself". In this game we will fight like the champions to show who is the best player in each match. With existing weapons, players must be able to set a strategy to win the match

This is a continuation of the popular game NoKa 2, since it was announced for release in December 2021. Not much different from its predecessor, but NoKa 3 has various features that will satisfy players and with better graphics.
